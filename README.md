# hackerrank-grind

Daily HackerRank problem solutions — one commit a day, building consistency and sharpening DSA fundamentals.

---

## Structure

```
hackerrank-grind/
├── python/
│   └── YYYY-MM-DD_problem-name.py
└── README.md
```

Each file is named by date and problem slug so the log is self-documenting.
